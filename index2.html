CSS

CSS Nedir? Neden Kullanılır?

CSS (Cascading Style Sheets), web sayfalarının görünümünü ve düzenini kontrol etmek için kullanılan bir stil dilidir. HTML ile birlikte kullanılır ve web sayfalarının tasarımını, renklerini, yazı tiplerini, boşluklarını ve diğer görsel öğelerini belirlemek için kullanılır. CSS, web sayfalarının daha çekici ve kullanıcı dostu olmasını sağlar.




METİN HİZALAMA

CSS ile metin hizalama işlemi "text-align" özelliği kullanılarak yapılır. Metin hizalama, bir metnin yatay olarak nasıl konumlandırılacağını belirler. İşte bazı yaygın metin hizalama seçenekleri:


1. Sol Hizalama (left): Metin, kapsayıcı öğenin sol kenarına hizalanır.
2. Sağ Hizalama (right): Metin, kapsayıcı öğenin sağ kenarına hizalanır.
3. Ortala (center): Metin, kapsayıcı öğenin ortasına hizalanır.
4. İki Yana Yasla (justify): Metin, kapsayıcı öğenin her iki kenarına yaslanır ve satırlar arasında boşluklar eklenir.




KENARLIK (BORDER)

CSS ile kenarlık eklemek için "border" özelliği kullanılır. Kenarlık, bir öğenin etrafına çizilen çizgidir ve çeşitli stillerde, kalınlıklarda ve renklerde olabilir. İşte kenarlık eklemek için bazı temel örnekler:


1. Kenarlık Stili: Kenarlık çizgisinin stilini belirler. Örnekler: solid (düz), dashed (kesikli), dotted (noktalı).
2. Kenarlık Kalınlığı: Kenarlık çizgisinin kalınlığını belirler. Örnekler: 1px, 2px, 5px.
3. Kenarlık Rengi: Kenarlık çizgisinin rengini belirler. Örnekler: red, #00ff00, rgb(0, 0, 255).
